Тестовое здание:

Необходимо написать скрипт на Python, PS, Ruby… любой другой скриптовый язык либо вообще приложение на .Net. Скрипт/приложение должен уметь:

1.       Проверять наличие изменений во всех ветках репозитория git@gitlab.com:kontur-tasks/trytobuild.git в заданный интервал времени (https://gitlab.com/kontur-tasks/trytobuild)

2.       При обнаружении изменений в ветке вытянуть изменения, собрать код.

3.       Запустить полученное приложение, сохранить вывод приложения в консоль.

4.       Запаковать приложение в nuget пакет. При этом в метаданных пакета должны быть:

a.       ID пакета (любое фиксированное значение),

b.       Version - (задание со звездочкой - версия, сформированная при сборке приложения из исходников)

c.       Описание - в описании необходимо отобразить вывод приложения в консоль + хеш последнего коммита.

d.       Автор - автор коммита.

1.       Полученный нугет пакет необходимо выложить (руками или, в качестве задания со звездочкой - из скрипта) на www.nuget.org (другой любой нугет хостинг).

2.       Дополнительно, как задание со звездочками: настройки, такие как URL репозитория, частота проверки, e-mail адрес для уведомлений, необходимо передавать либо как аргументы скрипту, либо вычитывать из конфигурационного файла.

Дополнительно, как задание с двумя звездочками: необходима обработка исключений + отправка на почту неудачных попыток проверки изменений/сборки проекта/запаковки в nuget пакет
